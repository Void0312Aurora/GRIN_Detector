# 基于物理信息神经网络和液晶微分的微透镜缺陷检测系统中期检查报告

- 项目名称：基于物理信息神经网络和液晶微分的微透镜缺陷检测系统
- 报告日期：2026年4月7日
- 报告阶段：中期检查
- 成熟度建议：`6（分析结果）`

## 一、项目摘要（200字以内）

面向微透镜阵列在光通信、成像与精密制造中的高精度在线检测需求，本项目聚焦无偏置液晶微分成像中相位符号丢失、积分伪影明显和真实标定困难等关键瓶颈，提出“差分物理模型 + 物理信息神经网络 + 质量门控”的研究路线。前期结果表明，该方法有望在降低硬件复杂度的同时提升缺陷定位能力和结果稳定性，为透明光学元件的低成本、高鲁棒检测提供新方案。

## 二、项目主要进展

### 1. 已完成研究问题聚焦与技术路线细化

项目围绕申报书提出的核心问题，进一步明确了研究对象和技术边界，即在无偏置液晶微分成像条件下，围绕“标准面 + 局部稀疏缺陷残差”的微透镜检测任务开展研究。当前已经形成了“差分物理建模 + 物理信息神经网络约束 + 结果质量控制”的总体技术路线，为后续实验和成果整理奠定了统一框架。

从科学问题上看，项目试图把无偏置条件下的符号恢复、奇点区域稳定重建和物理一致性约束纳入同一逆问题框架；从工程应用上看，项目希望用算法替代部分传统机械偏置和积分处理环节，降低系统复杂度，提升在线检测场景下的稳定性和可推广性。

### 2. 已完成算法原型构建与阶段性仿真验证

项目已完成无偏置相位恢复算法原型的实现，并围绕传统积分重建方法与所提方法开展了多组对比实验和消融分析。阶段结果表明，所提方法在缺陷识别能力、整体重建稳定性和边缘伪影抑制方面整体优于传统基线，初步证明了利用物理信息约束解决无偏置符号恢复问题的可行性。

在典型仿真场景中，所提方法对划痕、点状缺陷和较大缺陷均表现出较好的识别能力；同时，实验也暴露出超小缺陷和强噪声条件下的性能下降问题。这一结果说明项目主线方向是成立的，但距离稳定适配更复杂场景仍有进一步优化空间。

### 3. 已完成阶段性质量评价思路构建

考虑到后续真实实验场景中难以获得严格真值，项目已同步建立不完全依赖真值的质量评价思路，重点从物理一致性、边缘伪影和视场外泄漏等方面判断重建结果是否可信。该工作使项目评价方式从“只看误差指标”进一步推进到“同时关注结果可靠性和可落地性”，有助于后续真实样本筛选与质量控制。

### 4. 已开展预真实验证与鲁棒性分析

针对后续真实实验可能面临的参数偏移和分布变化，项目已围绕分辨率变化、噪声变化、孔径偏差和标定误差等因素开展了预真实验证。现有结果说明，方法对小幅参数偏差具有一定鲁棒性，但在高分辨率迁移、强噪声和明显域偏移条件下仍会出现性能退化。该部分工作为下一阶段的模型修正和实验设计提供了明确依据。

### 5. 已完成真实实验方案和数据规范准备

项目已完成 qDIC 单模态实验记录草案、qDIC 与白光干涉仪联合采集方案以及真实数据记录模板设计。当前阶段虽尚未形成完整真实样本闭环，但已经完成了样本组织、采集流程、配对关系、数据记录和后续验证路径的前期论证。这意味着项目已从单纯仿真研究推进到面向真实应用验证的准备阶段。

### 6. 阶段成果形式

截至目前，项目已形成的阶段成果主要包括：算法原型、仿真分析结果、阶段性质量评价方案、真实实验实施方案和中后期成果凝练基础材料。论文和专利尚处于整理与提炼阶段，暂未形成正式投送成果。

## 三、项目存在问题及解决方案

### 1. 真实实验闭环尚未完成

当前工作主要集中在理论分析、算法实现和仿真验证层面，尚未完成 qDIC/WLI 配对真实样本的正式采集、对准和定量验证。其主要原因在于，项目在中前期优先解决了方法是否成立、评价如何开展的问题，而真实实验还涉及仪器验收、采集规范、跨模态配准和样本设计等额外工作。

针对这一问题，下一阶段将先以 pilot 形式完成 clean repeat 和 paired defect 两类核心样本采集，优先跑通最小真实验证闭环，再逐步扩大样本规模。

### 2. 超小缺陷和复杂场景下的稳定性仍需提升

现有实验结果表明，模型在常见缺陷场景下已具备较好性能，但对超小缺陷、强噪声和明显域偏移条件仍较敏感。这说明项目当前已经具备“可行性验证”基础，但在达到更高灵敏度和更强泛化能力方面仍有差距。

下一阶段将结合困难样本增强、多尺度约束优化、参数鲁棒性分析以及真实样本反馈，进一步提升模型在小缺陷和复杂场景下的稳定性。

### 3. 质量评价阈值仍需用真实数据再校准

当前质量评价方案已经建立，但其统计依据主要仍来自仿真场景和预真实验证结果。随着真实样本接入，评价阈值仍需要结合 clean repeat 数据重新校准，否则可能影响对异常样本和失败重建的判断精度。

为此，项目后续将建立小规模真实验证集，对关键质量指标进行重新标定，并据此优化中后期实验评价标准。

### 4. 经费执行将主要集中在后续实验阶段

项目中期前的工作重点是方法设计、仿真验证和实验方案论证，因此经费支出高峰尚未完全到来。后续经费将主要用于真实样本采集、测试加工、实验耗材和必要的标定验证工作。

## 四、下一步工作计划

### 1. 继续优化算法主线

围绕超小缺陷、强噪声和尺度变化等薄弱环节继续开展针对性优化，进一步提升模型对弱缺陷的敏感性和对复杂条件的鲁棒性。

### 2. 完成真实实验 pilot 采集

按照既定方案优先开展 clean repeat 和 paired defect 实验，形成可用于真实验证的最小样本闭环，并完成基础元数据记录和质量检查。

### 3. 推进 qDIC 与 WLI 的配准和联合分析

在真实样本到位后，完成 clean 与 defect 状态下的跨模态对应、几何配准和弱真值构建，为项目后续定量分析提供支撑。

### 4. 重校质量评价标准

结合真实 clean 样本和重复采集结果，对质量评价阈值进行更新，使评价体系更贴近真实实验条件，提升后续结果筛选和异常识别能力。

### 5. 凝练论文和专利成果

在完成真实验证闭环后，系统整理项目方法、关键结果和创新点，形成论文初稿及专利技术点，为项目结题前的成果投送做好准备。
